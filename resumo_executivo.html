Sobre o Resumo Executivo

Origem dos dados Os números apresentados em cada slide são extraídos diretamente do painel interativo do município correspondente. Todos os slides incluem link de retorno para o painel para análise aprofundada.
Como apresentar Cada relatório é uma apresentação navegável. Use as setas ←/→ ou Espaço para avançar, F para tela cheia e P (ou o botão "Exportar PDF") para gerar versão imprimível. Esc abre o menu de slides.
Estrutura dos slides Cada apresentação cobre as 7 dimensões do Produto 2 do TR (demografia, emprego/renda, vulnerabilidade, infraestrutura, tecido econômico, academia, sociedade civil) e fecha com a síntese de adicionalidade do Produto 3.
